# Movie Rating & Box Office Analysis

## Project Overview
This is a lightweight data analytics project developed to analyze movie ratings and box office collections across different cinematic genres using Python.

## Key Insights discovered:
* **Highest Rated Genre:** Action and Sci-Fi movies consistently receive higher audience and critic ratings compared to standard romance genres.
* **Box Office Leaders:** Sci-Fi movies show the highest revenue potential, heavily driven by titles like *Avatar* and *Inception*.
* **Rating Patterns:** Movies with a rating above 8.5 show a strong correlation with higher global box office returns.

## Tech Stack Used:
* **Language:** Python 3
* **Libraries:** Pandas, Matplotlib, Seaborn

## How to Run:
1. Clone this repository.
2. Open `movie_analysis.ipynb` in VS Code or Jupyter Notebook.
3. Run all cells to view the visualization charts.
